# Munich AI Nexus – State: Persistence for Resilience
**Date:** 2026-05-21  
**Location:** Celonis, Munich  

## Theme
*Stateful agentic pipelines* – How do you build AI systems that remember, recover, and replay?  

## Agenda
- **Act I: Agents memory and learning**  
  Speaker: [Hande Kafka], [David Louis Hollembaek]  
  Summary: State as knowledge / memory layer.  
  Focus:
    - memory persistence
    - retrieval + evolving state
    - graph / structured memory
    - long-term agent memory

- **Act II: How we control and validate state**  
  Speaker: [Dr. Simon Stieber]  
  Summary: State as control & evaluation layer.  
  Focus:
    - test harnesses
    - reproducibility
    - context lifecycle
    - state as validation artifact

- **Act III: Production-Readiness Challenge**  
  Task: Design a resumable agent workflow that survives failure
  Format: Live collaborative review of your submissions.
